# NhlGoalWatcher

Watches NHL for goal score updates, and triggers events

## Installation

Add this line to your application's Gemfile:

    gem 'nhl_goal_watcher'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nhl_goal_watcher
